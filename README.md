# Skin Cancer & Health Equity — Cleaning, EDA & Analysis

Graduate data-analysis project — Emory MPH, BIOS 584.

## What it does
Cleans a skin-cancer dataset, engineers a race/ethnicity grouping (White vs. POC), and tests health-equity questions — for example, whether the number of primary malignancies differs between POC and White patients — using grouped summary statistics, boxplots, and chi-square tests. Analysis logic is factored into reusable functions.

## Tools
Python · pandas · numpy · scipy · matplotlib
